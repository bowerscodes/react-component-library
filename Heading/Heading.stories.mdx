# Heading

A `<Heading/>` component renders a string of text inside a `<h1>` tag, with
associated styling. 

<Canvas>
  <Story name="Default">
    <Heading>Heading</Heading>
  </Story>
</Canvas>

## `hNumber` prop

An optional `hNumber` prop of `1 | 2 | 3 | 4 | 5 | 6` can be passed in to change the
heading level, otherwise the Heading component will default to rendering the 
headingText in an `<h2>` tag.

<Canvas>
  <Story name="hNumber">
    <Heading hNumber={1}>Heading 1</Heading>
    <Heading hNumber={2}>Heading 2</Heading>
    <Heading hNumber={3}>Heading 3</Heading>
    <Heading hNumber={4}>Heading 4</Heading>
    <Heading hNumber={5}>Heading 5</Heading>
    <Heading hNumber={6}>Heading 6</Heading>
  </Story>
</Canvas>

## Alternative sizes
An optional `size` prop of `'s' | 'm' | 'l' | 'xl'` can be passed in to change the 
size, otherwise it is defaulted to `'m'`.

<Canvas>
  <Story name="Alternative Sizes">
    <Heading size="s">Small</Heading>
    <Heading size="m">Medium</Heading>
    <Heading size="l">Large</Heading>
    <Heading size="xl">Extra Large</Heading>
  </Story>
</Canvas>
