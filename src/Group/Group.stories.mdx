# Group

A Group can be used to organise form fields passed in as `children`.
Groups can be set to wrap their fields in a `<fieldset>` tag with 
associated `<legend>`, by passing the optional `fieldset` prop.

<Canvas>
  <Story name='Default'>
    <Group id='group' label='Basic Group' fieldset>
      <TextInput />
    </Group>
  </Story>
</Canvas>

## Group with Hint & Error 

Groups act as an enclosing wrapper which handles associated Hint and Error messages.

<Canvas>
  <Story name='With Hint and Error'>
    <Group 
      id='group' 
      label='Enter your address' 
      fieldset 
      hint='Addresses must be based in the UK' 
      error='Please enter a valid address'
    >
      <TextInput error/>
      <TextInput error/>
      <TextInput error/>
    </Group>
  </Story>
</Canvas>
