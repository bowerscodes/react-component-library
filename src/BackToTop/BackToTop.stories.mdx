# Back To Top

The `<BackToTop/>` component is a link which, by default, appears once the 
user has vertically scrolled down the page by a set value. When clicked, it 
scrolls the user back to the top of the page.

To override the default behaviour, the optional boolean `overrideShow` or 
numeric `showOnScrollY` props can be passed in.
