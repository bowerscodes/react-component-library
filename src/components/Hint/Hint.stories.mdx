# Hint

A `<Hint>` is an indented text string which provides additional information 
about something.

<Canvas>
  <Story name='Default'>
    <Hint>This is a Hint.</Hint>
  </Story>
</Canvas>

## Alternative sizes

The `<Hint>` component can be rendered in three different sizes: `'s' | 'm' | 'l'`.

<Canvas>
  <Story name='Alternative Sizes'>
    <Hint size='s'>This is a small Hint.</Hint>
    <Hint size='m'>This is a medium Hint.</Hint>
    <Hint size='l'>This is a large Hint.</Hint>
  </Story>
</Canvas>
