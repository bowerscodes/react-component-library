# Button

The `<Button/>` component renders a clickable `<button>` element, which 
accepts an `onClick` handler and a `type` prop to determine the button's 
appearance. Available types are `primary`, `secondary` and `warning`.

The button's text can be passed in either as a `label` prop, or as a child - 
i.e. `<Button>Click me</Button>`.

## Overriding the Button size

Buttons will automatically resize to fit the contents of their `label` or 
children by default. This can be overridden by passing in a value for the `size` prop. 
Available sizes are `'xs'`, `'s'`, `'m'`, `'l'`, and `'xl'`.
