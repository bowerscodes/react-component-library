# TextArea

A `<TextArea/>` component is used when the user is required to enter more 
than one line of text.

## Alternative Sizes

A TextArea will take up 100% of the width of its container by default.
This can be overridden by passing a value of 
`'one-third' | 'two-thirds' | 'two-thirds'` to the `width` prop.
