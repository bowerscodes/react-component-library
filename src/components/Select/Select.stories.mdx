# Select

A `<Select/>` component should be used when a user is required to select 
<strong>one</strong> option from a longer list of options.

<Canvas>
  <Story name='Default'>
    <Select

## With Group

Combining a Select component with a `<Group>` enables a `<Label>`, `<Hint>` and 
`error` message to be added.

<Canvas>
  <Story name='With Group'>
    <Group 
      id='select-group'
      label='Please select your Birth Month'
      hint='Your selection may be changed at any time.'
      error='Please select a month'

      <Select 
        id='select'
