# Radios

`<Radios>` should be used when a user needs to select <strong>one</strong> option from a list.

<Canvas>
  <Story name='Radios'>
    <Group id='radios-group' label='Would you like us to keep you updated via email?'>
      <Radios 
        id={'example'}
        fieldId={'example'}
        options={[
          { label: 'Yes please!', value: 'yes' },
          { label: 'No Thanks', value: 'no' },
        ]}
        error
      />
    </Group>
  </Story>
</Canvas>

## With Hint and Error

Rendering a `<Radios/>` component inside a `<Group>` enables a `label`, 
optional `hint` and `error` message to be displayed.

<Canvas>
  <Story name='With Hint & Error'>
    <Group id='radios-group' label='Would you like us to keep you updated via email?' hint='We will only send you relevant information.' error='Please select an option.'>
      <Radios 
        id={'example'}
        fieldId={'example'}
        options={[
          { label: 'Yes please!', value: 'yes' },
          { label: 'No Thanks', value: 'no' },
        ]}
      />
    </Group>
  </Story>
</Canvas>
