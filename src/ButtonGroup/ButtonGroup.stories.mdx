# Button Group

A `<ButtonGroup/>` is a set of buttons that are visually grouped together 
along with a `<Label/>` and optional text and `<Hint/>`. It is typically 
used to group related actions. 
Typical Button Group configurations are shown below.

## Single Button
<Canvas>
  <Story name='Single Button'>
    <ButtonGroup 
      id='single-button'
      label='Before you begin'
      buttons={[{ children: 'Start', type: 'primary' }]}
      content={{ 
        text: 'Please ensure you have all the required information to hand.' ,
        hint: 'This is form will take around 5 minutes to complete' 
      }}
      width='two-thirds'
    />
  </Story>
</Canvas>

## Two Buttons

<Canvas>
  <Story name='Two Buttons'>
    <ButtonGroup 
      id='two-buttons'
      label='Confirmation'
      buttons={[
        { children: 'Continue', type: 'primary' },
        { children: 'Cancel', type: 'secondary' }
      ]}
      content={{ 
        text: 'Please ensure the information you have provided is accurate.' ,
        hint: 'Responses cannot be amended after submission' 
      }}
      width='full'
    />
  </Story>
</Canvas>
